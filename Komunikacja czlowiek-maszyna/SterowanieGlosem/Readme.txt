Program pozwala na realizacje następujących komend:
-otwórz - otwiera stronę z możliwością rysowania
-kliknij
-trzymaj - trzyma lewy przycisk myszy, aby móc rysować
-puść - puszcza lewy przycisk myszy

poprawne użycie kierunków:
- Należy wybrać kierunek np. Lewo, następnie zaczekać, aż bot poprosi o ponowne powiedzenie komendy. Należy teraz podać ilość o jaką chcemy przesunąć kursor.
Przykładowe użycia: Lewo, zaczekaj do kolejnego nasłuchu 50
		    Prawo góra, zaczekać, 250
		    kierunek, odczekać, ilość

Aby program poprawnie działał należy pamiętać o zmianie ścieżki do geckodriver.exe
